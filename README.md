WebSpadmin (Spa Administration System)

WebSpadmin adalah sistem informasi pengelolaan administrasi SPA berbasis web yang dirancang untuk membantu efisiensi operasional manajemen spa, mulai dari pengelolaan reservasi pelanggan, transaksi pembayaran, hingga manajemen data master seperti layanan, ruangan, terapis, dan ulasan pelanggan.

Proyek ini dikembangkan untuk memenuhi tugas besar mata kuliah ISB-202 Rekayasa Perangkat Lunak (RPL).
